Для локального запуска веб-приложения:
 1. Установите Python 3.12 и PostgreSQL.
 2. Создайте базу данных PostgreSQL с названием dbae.
 3. Корневую папку проекта AETY откройте в терминале.
 4. Создайте виртуальное окружение командой:
    python -m venv .venv
 5. Активируйте виртуальное окружение:
    .venv\Scripts\activate
 6. Установите зависимости:
    pip install -r requirements.txt
 7. Скопируйте файл .env.example в .env и укажите настройки базы данных, почты и секретный ключ.
 8. Выполните миграции базы данных:
    python manage.py migrate
 9. При необходимости создайте администратора:
    python manage.py createsuperuser
 10. Запустите сервер:
    python manage.py runserver
 11. Вставьте ссылку в URL строку браузера:
    http://127.0.0.1:8000/

Для запуска локального сервера с доступом с других устройств в одной сети используйте команду:
python manage.py runserver 0.0.0.0:8000

После этого сайт можно открыть с другого устройства по адресу:
http://IP_КОМПЬЮТЕРА:8000/


Если при регистрации используется подтверждение почты, то для локальной разработки можно оставить console email backend. В этом случае код подтверждения будет выводиться в терминал, где запущен Django-сервер. Для реальной отправки писем необходимо указать SMTP-данные в файле .env.

После запуска можно проверить основные функции проекта:
 - регистрация и авторизация пользователей;
 - просмотр каталога онлайн-курсов;
 - подача заявки студентом на курс;
 - обработка заявок администратором;
 - управление курсами, группами, модулями, занятиями и расписанием;
 - добавление материалов, заданий и тестов преподавателем;
 - отправка ответов студентом;
 - автоматическая проверка тестов;
 - проверка заданий преподавателем;
 - просмотр уведомлений для разных ролей.
